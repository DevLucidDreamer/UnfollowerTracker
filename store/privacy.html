맞팔 체커 (Unfollower Checker) — 개인정보 처리방침

최종 업데이트: 2026-06-20

“맞팔 체커”(이하 “본 확장”)는 사용자의 개인정보를 수집·저장·전송하지 않습니다. 본 문서는 본 확장이 데이터를 어떻게 다루는지 투명하게 설명합니다.

1. 수집하는 정보

본 확장은 사용자의 이름, 이메일, 비밀번호, 위치, 결제정보 등 어떠한 개인정보도 수집하지 않습니다. 아이디·비밀번호를 입력받지 않으며, 이미 사용자가 로그인해 둔 instagram.com 브라우저 세션을 이용합니다.

2. 처리하는 데이터와 목적

사용자가 확장 아이콘을 눌러 “분석 시작”을 실행할 때에 한해, 사용자 본인 계정의 팔로잉/팔로워 목록(아이디, 이름, 프로필 사진 URL, 공개/인증 여부)을 instagram.com에서 조회합니다. 이 데이터는 “내가 팔로우하지만 나를 맞팔하지 않는 계정” 목록을 계산하는 단일 목적에만 사용됩니다.

3. 데이터의 저장 및 전송

  • 모든 처리는 사용자 브라우저 안에서만 일어납니다.
  • 본 확장은 어떠한 데이터도 외부 서버로 전송하지 않습니다. 개발자를 포함한 제3자가 사용자의 데이터에 접근할 수 없습니다.
  • 분석 결과는 메모리에서만 다뤄지며, 사용자가 직접 CSV 내보내기를 누른 경우에만 사용자 기기에 파일로 저장됩니다.
  • 데이터를 판매·대여·광고 목적으로 사용하지 않습니다.

4. 권한 사용

  • scripting / host_permissions (instagram.com): 사용자가 버튼을 눌렀을 때 instagram.com 탭에 수집 스크립트를 주입해 본인 데이터를 읽기 위해 사용합니다. 다른 사이트에는 접근하지 않습니다.

5. 제3자 서비스

본 확장은 분석 도구, 광고 SDK 등 어떠한 제3자 추적 서비스도 포함하지 않습니다.

6. 면책

본 확장은 비공식 도구이며 Instagram / Meta Platforms, Inc. 와 제휴·후원·승인 관계가 없습니다.
