# 맞팔 체커 (Unfollower Tracker)

내가 팔로우하지만 **나를 맞팔(맞팔로우)하지 않는** 인스타그램 계정을 빠르게 찾아주는 크롬 확장프로그램입니다.

JSON 파일을 내보내고 업로드할 필요 없이, **본인 인스타그램 세션에서 클릭 한 번**으로 분석합니다.

## 작동 방식

- 사용자가 열어둔 `instagram.com` 탭에 코드를 주입해, **그 탭의 로그인 세션**으로 본인의 팔로잉/팔로워 목록을 가져옵니다.
- 두 목록의 차집합을 계산해 "맞팔 안 하는 계정"을 보여줍니다.
- 모든 처리는 **사용자 브라우저 안에서만** 일어납니다. ID/PW를 받지 않고, 데이터를 외부 서버로 보내지 않습니다.

## 설치 (개발자 모드)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭 → 이 폴더(`UnfollowerTracker`) 선택

## 사용법

1. `instagram.com` 에 로그인한 상태로 탭을 엽니다.
2. 확장 아이콘 클릭 → **분석 시작**.
3. 팔로잉·팔로워 수집이 끝나면 맞팔 안 하는 계정 목록이 표시됩니다.
   - 아이디 클릭 시 프로필로 이동, 검색창으로 필터, **CSV 내보내기** 가능.

> 분석 중에는 팝업을 닫지 마세요(닫으면 진행이 중단됩니다). 팔로잉/팔로워가 많을수록 시간이 걸립니다(차단 회피를 위해 페이지마다 약간씩 대기).

## 📱 모바일(iOS/안드로이드)에서 쓰기

모바일 브라우저는 크롬 확장프로그램을 지원하지 않으므로, 같은 기능을 두 가지로 제공합니다.
둘 다 앱 설치 없이 동작하고, 모든 처리는 사용자 기기 안에서만 일어납니다.

빌드: `cd bookmarklet && node build.js` → 세 산출물 생성
- `shortcut-payload.js` : **iOS 단축어**에 붙여넣는 코드 (추천)
- `bookmarklet.txt` : `javascript:` 북마클릿 (폴백 · Android·데스크톱 공용)
- `install.html` : 위 두 방법의 단계별 설치 안내 페이지

소스는 `bookmarklet/src.js`(수집 + 결과 오버레이 UI) 하나이며, 단축어/북마클릿 모두 여기서 빌드됩니다.

### 방법 1 · iOS 단축어 (추천 · 가장 쉬움)

iPhone 기본 **단축어** 앱의 `웹페이지에서 JavaScript 실행` 액션은 **Safari 현재 탭의 페이지 컨텍스트**에서
실행되므로, 로그인된 instagram.com 세션을 그대로 사용합니다(북마클릿과 동일한 원리).

1. 단축어 앱 → 새 단축어 → **웹페이지에서 JavaScript 실행** 액션 추가
2. `shortcut-payload.js`(또는 `install.html`의 복사 버튼) 코드를 액션에 붙여넣기
3. 이름 `맞팔 체커`, **공유 시트에 표시** 켜기
4. 실행: `instagram.com` 로그인 탭에서 **공유 버튼 → 맞팔 체커** 탭 (첫 실행 시 1회 허용)

> 배포: 단축어 → 공유 → **iCloud 링크 복사**. 받는 사람은 링크 누르고 “단축어 추가” 한 번이면 끝.

### 방법 2 · 북마클릿 (폴백)

**iOS Safari 요약**
1. `install.html`에서 북마클릿 코드 복사
2. 아무 페이지에서 북마크 추가 → 이름 `맞팔 체커`
3. 북마크 편집 → URL을 지우고 복사한 코드 붙여넣기
4. `instagram.com` 로그인 상태에서 주소창에 `맞팔` 입력 → 북마크 실행

## 비공개 계정은?

- **본인 계정이 비공개여도 정상 동작합니다.** 본인 세션으로 본인 데이터를 읽기 때문입니다.
- 본인이 팔로우 중인 *상대*가 비공개여도 맞팔 여부 판정에는 문제 없습니다(상대가 내 팔로워 목록에 있는지만 보면 되므로).

## 주의 / 한계

- 인스타그램의 비공개 내부 API를 사용하므로 **이용약관상 자동 수집은 회색지대**입니다. 과도하게 자주 실행하지 마세요.
- 인스타그램이 응답 형식이나 엔드포인트를 바꾸면 동작이 깨질 수 있습니다.
- 짧은 시간에 너무 많이 호출하면 인스타그램이 일시적으로 차단(429)할 수 있습니다. 이때는 몇 분 뒤 다시 시도하세요.

## 아이콘 (선택)

`manifest.json`에 아이콘을 지정하지 않아 크롬 기본 아이콘이 표시됩니다. 원하면 `icons/` 폴더에 16/48/128px PNG를 넣고 manifest의 `action.default_icon`에 등록하세요.

## 파일 구조

| 파일 | 역할 |
|------|------|
| `manifest.json` | 확장 설정(MV3) |
| `popup.html` / `popup.css` | 팝업 UI |
| `popup.js` | 탭 탐색, 데이터 수집기 주입, 차집합 계산, 렌더링·CSV |
